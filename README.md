# MappApp

## Steps:
1. Clone the .env.template y rename it as .env
2. Fill the environment variables
3. Create the Angular Envs (optional)
```
npm run envs
```
4. For development environment run (include the Angular Envs command)
```
npm run start
```
5. For production environment run (include the Angular Envs command)
```
npm run build
```
